# Auth

Auth模块负责为应用程序指定基本交易和帐户类型

## 可用命令

| 名称        | 描述                 |
| ----------- | -------------------- |
| [account]() | 查询账户信息         |
| [params]()  | 查询当前Auth模块参数 |

## gnchaind query auth account

该命令用于查询账户信息。

```bash
gnchaind query auth account [address] [flags]
```

## gnchaind query auth params

该命令用于查询当前Auth模块参数。

```bash
gnchaind query auth params [flags]
```
